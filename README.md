# Descrizione
Owly Book Finder è un'applicazione web che permette di cercare libri per categoria tramite le API di Open Library (https://openlibrary.org/developers/api)
Una volta caricati i libri della categoria richiesta, è possibile cliccare sul titolo di un libro per visualizzarne la descrizione.
In caso l'applicazione non riesca a caricare i risultati della ricerca o la descrizione del libro, verrà visualizzato un messaggio di errore.
Tutte le funzioni dell'applicazione sono testate tramite Vitest
# Architettura
Il progetto segue il pattern MVC:
Model: api.js
View: ui.js
Controller: main.js
